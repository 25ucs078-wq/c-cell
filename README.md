# 🚀 C-CELL App

### Your Gateway to Campus Life at LNMIIT

C-CELL is a modern campus companion built to bring events, clubs, announcements and student life together into one seamless experience.

Designed with a premium UI and built for scalability, C-CELL aims to become the digital hub for the LNMIIT community.

---

## ✨ Features

🔐 Secure Login System

🎨 Modern Dark UI

⚡ Cinematic RED CORE Loading Experience

🎭 Event Showcase

🏛 Clubs & Societies

📢 Campus Updates

🖥 Responsive Web Experience

☁ Powered by Supabase

🚀 Built with Flutter
